# Sudoku Console Edition

Proyecto desarrollado en lenguaje C (C17).

## Descripción

Sudoku Console Edition es un videojuego de Sudoku para consola desarrollado en lenguaje C. El proyecto implementa una arquitectura modular, sistema de usuarios, guardado automático, historial de partidas, ranking y generación automática de tableros de Sudoku.

## Características

- Registro e inicio de sesión de usuarios.
- Guardado automático de partidas.
- Generación aleatoria de Sudokus.
- Tres niveles de dificultad.
- Cronómetro de juego.
- Ranking de jugadores.
- Historial de partidas.
- Interfaz interactiva en consola.
- Compatible con Linux y Windows.

## Tecnologías

- Lenguaje C (C17)
- GCC
- Git
- GitHub
- Visual Studio Code

## Estructura del proyecto

- `src/` Código fuente.
- `include/` Archivos de cabecera.
- `data/` Archivos de datos.
- `docs/` Documentación.
- `obj/` Archivos objeto.
- `bin/` Ejecutables.
